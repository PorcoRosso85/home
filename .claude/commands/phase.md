# phase
/phase

# 説明
タスクをBaby Steps（15-30分単位）に分解し、各フェーズの最後にテスト実行を含むタスクリストをTodoWriteで作成する。

# 実行内容
1. 目標を小さなフェーズに分解
2. 各フェーズの最後にテスト実行ステップを自動追加
3. TodoWriteツールでタスクリストを作成
4. 依存関係を考慮した優先度設定

# テスト自動追加ルール
- 実装: `nix run .#test`実行タスクを追加
- 設計: レビュー完了確認タスクを追加
- 設定: ヘルスチェックタスクを追加
- ドキュメント: リンクチェックタスクを追加

# 連携
- TodoReadで進捗確認
- `/orch`: 作成されたタスクの並列実行
- `/tdd`: 各実装フェーズでTDD適用
