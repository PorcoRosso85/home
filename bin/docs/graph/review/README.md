# graph/review

## 責務

このディレクトリの責務は、親ディレクトリ（`../`）である `docs/graph/` のレビューを行うことです。

## 概要

`docs/graph/` ディレクトリに実装されたグラフ関連の機能、設計、実装に対して以下のレビュー活動を実施します：

- 実装の妥当性検証
- テストカバレッジの確認
- パフォーマンス評価
- 設計の一貫性確認
- ドキュメントの充実度評価

## ディレクトリ構造

```
docs/graph/          # レビュー対象
└── review/          # このディレクトリ（レビュー実施）
    └── README.md    # 責務の定義（このファイル）
```
