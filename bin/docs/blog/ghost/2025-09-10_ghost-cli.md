# Ghost CLI

Ghost CLI is a powerful command-line tool designed to help developers install, configure, and manage Ghost publishing platforms.

## Main Commands

### Installation & Setup
- `ghost install` - Installs Ghost with a single command
- `ghost config` - Configures Ghost settings  
- `ghost setup` - Configures server environment (MySQL, NGINX, systemd)

### Instance Management
- `ghost start` - Starts Ghost instance
- `ghost stop` - Stops Ghost instance
- `ghost restart` - Restarts Ghost instance
- `ghost ls` - Lists all Ghost installations

### Maintenance
- `ghost update` - Upgrades Ghost to latest version
- `ghost backup` - Creates site data backup
- `ghost doctor` - Checks system compatibility and diagnoses issues
- `ghost log` - Views site logs

## Global Options

```bash
# Output usage information
ghost --help, ghost -h, ghost help, ghost [command] --help

# Enable verbose logging for debugging
ghost --verbose, ghost -V

# Print CLI and Ghost version
ghost --version, ghost -v, ghost version

# Run command in different directory
ghost --dir path/to/directory

# Run without interactive prompts
ghost --no-prompt

# Run without colors
ghost --no-color
```

## Key Features

### Installation
- Supports both production and development environments
- Configures MySQL database
- Sets up NGINX reverse proxy
- Configures systemd for process management
- Creates logical folder structure
- Handles user permissions automatically
- Provides SSL configuration support

### Environment Support
The CLI handles different environments:
- **Production**: Full setup with NGINX, MySQL, systemd
- **Development**: Simplified setup for local development

### Folder Structure
Ghost CLI creates and manages a logical folder structure for Ghost installations, keeping configuration, content, and system files organized.

## Benefits
- Simplifies Ghost installation and maintenance
- Reduces setup time from hours to minutes
- Provides consistent configuration across environments
- Automates common administrative tasks
- Ensures best practices for production deployments
