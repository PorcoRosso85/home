# データ照会言語（DQL）

要件グラフから情報を取得する操作群。

## 操作種別

### SEARCH
類似度ベースの検索。VSS/FTSハイブリッド方式。

### GET
単一要件の詳細取得。関連情報を含む。

### LIST
条件に基づく一覧取得。ページネーション対応。

## 主要テンプレート

- `search_requirements`: 類似要件検索（k近傍）
- `get_requirement`: ID指定での詳細取得
- `list_dependencies`: 依存関係グラフの取得

## 検索戦略

- **VSS**: 意味的類似性による検索
- **FTS**: キーワードマッチング
- **ハイブリッド**: 両方式の統合結果
