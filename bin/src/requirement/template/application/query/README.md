# クエリテンプレート

[アプリケーション層](../README.md)のユースケースを実現するクエリ定義群。

## 分類原則

### [`dml/`](./dml/README.md) - データ操作言語
要件グラフを変更する操作。作成・更新・削除。

### [`dql/`](./dql/README.md) - データ照会言語  
要件グラフから情報を取得する操作。検索・一覧・詳細。

## テンプレートの役割

- **パラメータ化**: 安全な値の注入
- **検証**: 入力値の妥当性確認
- **標準化**: 一貫性のある操作体系

## 命名規則

- `create_*`: 新規作成操作
- `update_*`: 既存データ更新
- `search_*`: 条件検索
- `get_*`: 単一取得
- `list_*`: 一覧取得

## ドメインルールの適用

各テンプレートは[ドメイン層](../../domain/README.md)で定義された制約を遵守。
