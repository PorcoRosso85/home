# アプリケーション層

[ドメインルール](../domain/README.md)に基づく操作可能なユースケースを定義。

## ユースケース分類

### 要件作成
重複チェックを伴う要件の新規作成。類似要件が存在する場合は通知。

### 要件検索
類似度ベースで関連要件を発見。VSS/FTSハイブリッド検索を活用。

### 依存管理
グラフ構造の操作。循環参照や深さ制限を自動検証。

## テンプレート体系

入力の標準化と検証を保証する操作定義。詳細は[`query/`](./query/README.md)を参照。

## フィードバックループ

1. **提案**: テンプレート+パラメータで操作を要求
2. **評価**: ドメインルールと類似検索による検証
3. **改善**: フィードバックに基づく修正

## 下位層との協調

[`infrastructure層`](../infrastructure/README.md)の技術基盤を利用してユースケースを実現。
