## 初回認証時の流れ

```bash
$ nix run .#run

🔐 初回認証を開始します...

🌐 ブラウザを開いています...

⏳ 認証を待っています...

✅ 認証成功！

（ブラウザで）
┌─────────────────────────────────────┐
│ Googleアカウントでログイン          │
│                                     │
│ メールアドレス: user@gmail.com      │
│ パスワード: ********                │
│                                     │
│ [次へ]                              │
└─────────────────────────────────────┘

↓ 認証後自動的に

┌─────────────────────────────────────┐
│ ✅ 認証成功！                       │
│                                     │
│ このタブは閉じて構いません。        │
└─────────────────────────────────────┘

✅ 認証完了！メールを取得します...

（以降、通常のメール表示）
```

## 主な特徴

1. **完全自動認証**: 初回認証時もブラウザが自動で開き、認証コードも自動取得
2. **トークン永続化**: 2回目以降は認証不要
3. **見やすい表示**: 絵文字を使った視覚的な表示
4. **フィルタ機能**: 未読メールのみの表示が可能
