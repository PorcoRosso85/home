# 適格請求書発行事業者公表システムWeb-API機能

## アプリケーションID発行条件

### 基本要件
- **費用**: 無料
- **利用資格**: 個人・法人問わず誰でも申請可能

### 必要な手続き（インボイスWeb-API利用の場合）

1. **Web-API利用規約への同意**
   - 必須事項

2. **アプリケーションID発行届出**
   - オンラインフォームで仮登録
   - メールで本登録URL受信
   - 本登録フォームで情報入力

3. **発行申請書の提出**
   - 「適格請求書発行事業者公表システムWeb-API機能アプリケーションID発行申請書」
   - 添付資料の提出が必要

4. **提出する情報**
   
   **個人の場合:**
   - 氏名
   - 氏名（カナ）
   - 電話番号
   - メールアドレス
   
   **法人の場合:**
   - 法人番号
   - 法人名
   - 担当部署名
   - 担当者氏名
   - 担当者氏名（カナ）
   - 電話番号
   - メールアドレス

### 法人番号システムWeb-APIとの関係

- **インボイスWeb-APIのIDを取得した場合**: 法人番号システムWeb-APIも同じIDで利用可能
- **法人番号システムWeb-APIのみのIDを持っている場合**: インボイスWeb-APIを利用するには追加申請が必要（令和5年1月20日以降）

### 発行期間
- 申請書類提出後、審査・承認プロセスあり
- 法人番号システムWeb-APIは2週間～1か月程度

### 提出先
- **メールアドレス**: invoice-web-api@nta.go.jp
- **担当部署**: 国税庁軽減税率・インボイス制度対応室

### 利用可能な機能（Ver.1）
1. 登録番号を指定して情報を取得
2. 取得期間を指定して更新（差分）情報を取得
3. 登録番号と日付を指定して情報を取得

### 注意事項
- REST方式のWeb-API
- 検証環境も提供（アプリケーションID必要）
- サービス公開時は取得元の明示が必要：「このサービスは、国税庁適格請求書発行事業者公表システムのWeb-API機能を利用して取得した情報をもとに作成しているが、サービスの内容は国税庁によって保証されたものではない」
