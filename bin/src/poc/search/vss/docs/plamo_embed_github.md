# plamo-embed GitHub Repository

## 概要
plamo-embedは、PLaMo-Embedding-1Bモデルを使用したPythonプロジェクトです。

## リポジトリ情報
- **URL**: https://github.com/shuntaka9576/plamo-embed
- **言語**: Python (98.6%)
- **作者**: shuntaka9576

## 実行方法

### 基本的な実行コマンド
```bash
# 依存関係の同期
uv sync

# メインスクリプトの実行
uv run main.py

# Makefileを使用した実行
make exec
```

## プロジェクト構成

### 主要ファイル
- `main.py`: メインスクリプト
- `blog_search_demo.py`: ブログ検索デモの実装
- `pyproject.toml`: プロジェクト設定ファイル
- `Makefile`: ビルド・実行コマンドの定義
- `uv.lock`: 依存関係のロックファイル
- `.python-version`: Pythonバージョン指定
