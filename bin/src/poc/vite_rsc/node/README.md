# Vite RSC POC

## 責務

このディレクトリは以下の確認試験を行うPOC（Proof of Concept）環境です：

- **vitejs/plugin-rsc** - Viteの公式React Server Componentsプラグインの検証
- **Node.js** - JavaScriptランタイムとしてNode.jsを使用した環境構築
- **Server Components SSR** - サーバーコンポーネントでSSR（Server-Side Rendering）提供することの確認試験

## 目的

React Server Components（RSC）をViteとNode.jsの組み合わせで動作させ、実用性と性能を検証する。

## 構成

- `flake.nix` - Nix開発環境定義（現在は空ファイル）
