# Waku Node RSC Implementation

このディレクトリの責務:

- **フレームワーク**: Waku (React Server Components最小実装)
- **ランタイム**: Node.js 22
- **デプロイ先**: Cloudflare Workers (参照: ../deploy/cloudflare)

## 概要

WakuフレームワークによるRSC実装の検証環境です。Node.js 22で動作し、Cloudflare Workersへのデプロイが可能です。
