# 基本エラーパターン定義

## 1. 型の不一致 (Type Mismatch)
- **説明**: 変数に宣言された型と異なる型の値を代入
- **期待**: 静的型チェッカーがエラーを検出
- **例**: 文字列型の変数に数値を代入

## 2. null/undefined アクセス (Null Access)
- **説明**: null または undefined の可能性がある値のプロパティ/メソッドにアクセス
- **期待**: 静的型チェッカーが潜在的なnullエラーを検出
- **例**: Optional型の値に対する無防備なアクセス

## 3. 存在しないプロパティ (Unknown Property)
- **説明**: オブジェクトに存在しないプロパティへのアクセス
- **期待**: 静的型チェッカーがプロパティの不在を検出
- **例**: 定義されていないフィールドの参照

## 4. 関数の引数不足 (Missing Arguments)
- **説明**: 関数呼び出し時に必要な引数を省略
- **期待**: 静的型チェッカーが引数の不足を検出
- **例**: 2引数の関数を1引数で呼び出し

## 5. 読み取り専用への代入 (Readonly Assignment)
- **説明**: readonly/const として宣言されたプロパティへの再代入
- **期待**: 静的型チェッカーが不変性違反を検出
- **例**: readonlyプロパティの変更試行
