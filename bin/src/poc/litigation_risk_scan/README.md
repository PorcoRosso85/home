# Litigation Risk Scanner

## これは何か

EDGARの提出書類と過去の集団訴訟パターンを突き合わせ、「この記述、3年前にあの会社が訴えられた時と87%一致」という警告を出すコンソールアプリ。

## 売るもの

**恐怖**: 「この一文を見逃したら、お前のキャリアが終わる」

## 真のMVP

```
litigation_risk_scan/
├── README.md       # このファイル
├── main.ts        # すべてのロジック（これだけ）
├── patterns.json  # 手動でコピペした危険パターン100個
└── .env           # EDGAR APIキー（もし必要なら）
```

以上。ディレクトリ構造なんてクソ食らえ。

## 実装ステップ

### ステップ1: main.tsを作れ（今すぐ）
- EDGARからデータを引っこ抜く
- patterns.jsonと突き合わせる
- 結果をコンソールに表示する
- **以上だ**

### ステップ2: 最初の信者を見つけろ
- ジュニアアナリスト1人にコンタクト
- 毎日手動でスクリプトを実行
- 結果をGmailからコピペして送信

### ステップ3: カネを奪え
- 「来月から月500ドル。銀行振込な」
- 請求書は手書きでOK
