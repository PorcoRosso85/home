# RGL - Requirement Graph Logic

開発決定事項管理ツール。決定の重複を防ぎ、過去の決定との関連性を発見し、決定の品質を向上させます。

## 使い方

このディレクトリで実行：

```bash
python run.py
```

指示に従って`uv run python run.py [コマンド]`を実行してください。
