
| 特徴             | 型クラス (Lean)                                 | オブジェクト指向の継承 (例: Java, C++)           |
| ---------------- | ----------------------------------------------- | ------------------------------------------------ |
| **目的**         | 行動 (振る舞い) の抽象化                          | 状態と行動 (実装) の共有と特殊化                  |
| **メカニズム**     | 型制約 (type constraints) とインスタンス (instances) | クラス (class) とサブクラス (subclass) の階層構造 |
| **多態性**       | 静的な多態性 (static polymorphism)               | 動的な多態性 (dynamic polymorphism)               |
| **コードの再利用** | 行動の抽象化による再利用                          | 実装の継承による再利用                             |
| **結合度**       | 疎結合 (loose coupling)                           | 密結合 (tight coupling)                            |
| **拡張性**       | 型クラスのインスタンスを定義することで拡張可能      | クラス階層を拡張することで拡張可能                  |

| 特徴             | 型クラス (Lean)                                 | `interface` (TypeScript)                              |
| ---------------- | ----------------------------------------------- | ----------------------------------------------------- |
| **主な役割**     | 行動 (振る舞い) の抽象化、多態性の実現              | オブジェクトの型 (shape) の定義、型の制約               |
| **焦点**         | 型が持つべき「能力」「性質」                        | オブジェクトが持つべき「プロパティ」「メソッド」           |
| **型推論**       | 強力な型推論                                     | 構造的な型付け (structural typing)                     |
| **インスタンス**   | 型クラスのインスタンス (明示的な実装)             | インターフェースを「実装する」 (暗黙的な構造的適合)       |


| 特徴             | 型クラス (Lean)                                 | `struct impl` (Rust)                                |
| ---------------- | ----------------------------------------------- | --------------------------------------------------- |
| **主な役割**     | 行動 (振る舞い) の抽象化、多態性の実現              | 構造体に関連するメソッド定義、トレイトの実装          |
| **関連付け**     | 型と行動の関連付け (型クラスとインスタンス)         | 構造体とメソッド/トレイト実装の関連付け             |
| **型推論**       | 強力な型推論                                     | 型推論 (型注釈が必要な場合もある)                   |
| **柔軟性**       | 高い (独立した定義と実装、型推論)                 | 比較的高い (構造体中心、トレイト境界による制約)       |
