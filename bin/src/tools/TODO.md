## Claude Desktop設定ビルダーツール

- 以下の機能を含める:
  - 現在の設定ファイルの読み込み

  - コマンドやパラメータの更新
  - ツールの許可リスト編集
  - 環境変数設定
  - JSON形式の検証
  - 設定ファイルの保存

### ビルド方法
- ツール実装後は、Argcfileが存在するディレクトリ（通常は `/home/nixos/bin/src`）で `argc build` を実行
- argcがインストールされていない場合は `nix run nixpkgs#argc -- build` コマンドを使用
- ビルド後は `argc check` で問題がないか確認（もしくは `nix run nixpkgs#argc -- check`）
- **注意**: 直接ビルドは行わないこと（実装のみ完了させる）
